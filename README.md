🧠 Syntax Surgeon — Grammar & Syntax Correction (Google Colab Project)

A simple NLP mini‑project that detects and corrects grammar, spelling, and basic syntax errors in English sentences using Python. Designed to run easily in Google Colab with minimal setup.

✨ Features

✅ Grammar mistake detection

✅ Automatic sentence correction

✅ Verb tense fixes

✅ Spelling correction

✅ Beginner‑friendly implementation

✅ Works fully in Google Colab (no local setup needed)

🛠️ Tech Stack

Python 3

language-tool-python (Grammar & syntax checking)

pyspellchecker (Optional spelling correction)

Google Colab (Execution environment)
